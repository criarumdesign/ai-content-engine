* n8n
* OpenAI
* Google Sheets
* GitHub

* Etsy listing generation
* Pinterest SEO content generation
* Instagram content generation
* Modular prompt architecture
* Multi-product scalability
* JSON structured outputs

* prompts → AI instructions
* workflows → n8n automations
* sheets → Google Sheets structure
* assets → hooks, CTAs and creative resources
