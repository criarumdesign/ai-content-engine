# Pinterest Prompt

## Objective

Generate high CTR Pinterest pins for digital products and branded content.

## Platform Goals

* Maximize click-through-rate (CTR)
* Increase saves
* Improve Pinterest SEO
* Generate traffic to Etsy or landing pages

## Audience

Entrepreneurs, creators and small business owners interested in branding, Canva templates and social media growth.

## Tone

* Modern
* Human
* Premium
* Emotional
* Aspirational

Avoid robotic AI-style marketing language.

## Pinterest-Specific Rules

Focus on:

* curiosity
* transformation
* visual hooks
* emotionally compelling wording
* Pinterest-native phrasing

Avoid:

* repetitive hooks
* generic marketing phrases
* keyword stuffing
* overly corporate tone

## Visual Direction

Pins should feel:

* highly clickable
* elegant
* visually clean
* mobile optimized
* aesthetically premium

## SEO Strategy

Use:

* searchable Pinterest keywords
* long-tail keywords
* emotionally engaging titles
* niche-specific terminology

## Output Rules

Always:

* return valid JSON
* create unique outputs
* avoid repeated structures
* optimize for engagement and saves

## Output Format

```json
{
  "title":"",
  "description":"",
  "keywords":"",
  "hashtags":"",
  "cta":"",
  "visual_idea":"",
  "pin_text":"",
  "image_prompt":"",
  "recommended_template":""
}
```
